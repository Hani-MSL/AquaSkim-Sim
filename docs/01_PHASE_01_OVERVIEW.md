# فاز 01 — اسکلت پروژه و تثبیت پارامترهای پایه

## هدف فاز
ایجاد یک محیط توسعهٔ تکرارپذیر در Windows/VS Code، تعریف قرارداد پوشه‌ها و ذخیرهٔ یک مجموعه پارامتر اولیه که تمام مراحل بعدی از آن استفاده خواهند کرد.

## خروجی‌های این Patch
- محیط Conda با نام `aquaskim-sim`
- ساختار کامل پوشه‌ها
- فایل پارامترهای پایه `config/base_parameters.yaml`
- چهار سناریوی اولیه
- ماژول پایتون برای خواندن و اعتبارسنجی تنظیمات
- دستور preflight
- تست‌های پایه
- اسکریپت‌های CMD

## معیار قبولی فاز
همهٔ فرمان‌های زیر باید بدون خطا اجرا شوند:

```bat
python -m aquaskim.cli preflight
python -m pytest -q
```

## خروجی مورد انتظار از کاربر
پس از اجرای دستورهای راه‌اندازی، متن کامل خروجی CMD شامل نسخهٔ Conda، خروجی preflight و نتیجهٔ pytest باید ارسال شود تا مرحلهٔ بعدی بر اساس وضعیت واقعی سیستم ساخته شود.
