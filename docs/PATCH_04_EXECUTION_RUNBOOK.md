# Patch 04 — راهنمای اجرای یک‌فرمانی

## فرمان تنها برای اجرای رسمی Patch 04

```bat
scripts\run_patch_04.bat
```

## کارهایی که این فرمان ثبت و اجرا می‌کند
1. فعال‌سازی محیط `aquaskim-sim`.
2. نصب editable پروژه از همان فولدر.
3. اجرای preflight.
4. بازتولید Phase 02 به‌عنوان وابستگی هندسه و جرم.
5. بازتولید Phase 03 به‌عنوان وابستگی آبخور و پایداری.
6. اجرای Phase 04.
7. اجرای کل آزمون‌ها.
8. ثبت `pip freeze`.
9. ذخیرهٔ stdout و stderr هر فرمان.
10. ذخیرهٔ Snapshot محیط و YAML تنظیمات.
11. هش SHA-256 تمام Artifactهای فازهای 02 تا 04.
12. کپی Snapshot خروجی‌ها.
13. ساخت Handoff رسمی.

## محل شواهد

```text
records/phases/phase_04/LATEST_RUN.md
```
