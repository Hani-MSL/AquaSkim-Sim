# Project overview

AquaSkim-Sim is a reproducible educational simulation workflow for a low-speed autonomous catamaran surface robot intended for floating-waste collection in sheltered water.

The public repository is organized around a clean rebuild: source code, configuration, tests and documentation are tracked; generated reports, figures, animations and delivery archives are recreated locally.

## Main capabilities

- low-speed 3-DOF motion simulation,
- reference mission execution,
- current-aware control and operating-envelope studies,
- payload and maneuver evidence generation,
- curated visual evidence,
- final English Word report generation,
- auditable final delivery ZIP generation.

## Scope boundary

The repository provides numerical simulation evidence only. It does not claim sea-trial certification, wave-response validation, onboard current-estimator validation or hardware commissioning.
