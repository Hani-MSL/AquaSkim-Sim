# Handoff from Phase 09 to Phase 10

Phase 09 provides the final numerical and visual validation package for the digital twin. Phase 10 will assemble the Word report and final delivery archive.

## Required inputs for Phase 10

1. Latest handoff from every Phase 02–09 directory.
2. Report-quality PNG and vector SVG files.
3. CSV tables and JSON summaries from all phases.
4. The Phase 09 scenario and Monte Carlo metrics.
5. The GIF/MP4 artefacts for the presentation appendix.
6. The complete reproducibility evidence tree.

## Phase 10 deliverables

- `AquaSkim-Sim_Final_Report.docx`
- PDF export if available locally
- final source-code archive
- final evidence archive
- final project manifest with SHA-256 checksums
- an executive one-page delivery summary
