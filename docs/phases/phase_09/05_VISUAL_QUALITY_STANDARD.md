# Phase 09 — Visual Quality Standard

## Required exports

Each static Phase 09 figure must be exported as:

1. PNG for Word insertion at a minimum raster size of `3000 × 1800 px`.
2. SVG for vector-quality reuse.

## Layout rules

- No dense parameter list may be written directly on a trajectory.
- Scenario paths, aggregate metrics and narrative interpretation must be placed in separate panels.
- All tables must use a dedicated table region with fixed columns.
- Informational cards must reserve separate vertical bands for title, primary value and note text.
- Empty sensitivity cells must be labelled `n=0`; interpolation is prohibited.

## Animation rules

- GIF and MP4 are required for presentation, not for quantitative measurement.
- The animation must use the same closed-loop mission traces stored in CSV.
- Static figures remain the authoritative visual source for numerical reporting.
