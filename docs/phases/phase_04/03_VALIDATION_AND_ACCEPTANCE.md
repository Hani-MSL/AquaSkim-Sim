# فاز 04 — اعتبارسنجی و معیارهای پذیرش

## آزمون‌های عددی

| آزمون | معیار |
|---|---|
| مقاومت در V=0 | دقیقاً صفر |
| مقاومت با سرعت | غیرکاهشی |
| Reynolds | با سرعت افزایش یابد |
| رانش پیشران | با مربع RPM افزایش یابد |
| توان پیشران | با مکعب RPM افزایش یابد |
| نقطهٔ هدف | رانش لازم کمتر از رانش حداکثر باشد |
| ذخیرهٔ رانش هدف | حداقل مقدار تعریف‌شده در YAML |
| جریان مخالف 0.20 m/s | در سرعت مأموریت قابل‌دستیابی باشد |
| سرعت تئوریک | بیشتر از حد سرعت عملیاتی 0.60 m/s باشد |
| سازگاری kT | رانش محاسبه‌شده در RPM حداکثر با ورودی حداکثر سازگار باشد |

## کنترل بصری
هر شکل باید هم PNG با حداقل 4500×2400 پیکسل و هم SVG داشته باشد. متن‌های توضیحی از نمودارهای اصلی جدا هستند و در پنل‌های ثابت رسم می‌شوند.
