# تحویل Phase 04 به Phase 05

## ورودی‌های آماده برای مدل انرژی
- توان پیشران در نقاط کاری مشخص.
- توان پیشران در جریان مخالف.
- توان تجهیزات ثابت (`hotel_load_w`).
- سرعت عملیاتی توصیه‌شده و ذخیرهٔ رانش.
- جدول‌های CSV کامل برای محاسبهٔ انتگرال انرژی مأموریت.

## محدودیت‌هایی که Phase 05 باید حفظ کند
- توان‌های Phase 04 برآورد اولیه هستند، نه منحنی آزمایشگاهی موتور/پروانه.
- مدل انرژی باید SOC، راندمان، ظرفیت قابل استفاده و شرایط بازگشت را اضافه کند.
