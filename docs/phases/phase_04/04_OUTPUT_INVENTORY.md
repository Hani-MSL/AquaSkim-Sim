# فاز 04 — موجودی خروجی‌ها

## شکل‌ها
- `phase04_resistance_dashboard`: تفکیک مقاومت‌ها و شاخص‌های ITTC.
- `phase04_propulsion_envelope`: رانش و توان برحسب RPM.
- `phase04_current_penalty`: اثر جریان مخالف بر رانش، throttle و توان.
- `phase04_operating_envelope`: فرمان کروز و ذخیرهٔ شتاب طولی.

## جدول‌ها
- منحنی مقاومت.
- منحنی پیشران.
- نقاط کاری کلیدی.
- تحلیل حساسیت جریان.
- معیارهای پذیرش.
