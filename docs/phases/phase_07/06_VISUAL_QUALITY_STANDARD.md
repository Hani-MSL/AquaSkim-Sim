# Phase 07 — Visual Quality Standard

- Minimum output raster size: 3000 × 1800 pixels.
- Every report figure also has an SVG version.
- Main geometry and data plots remain clear of long text blocks.
- Long engineering explanations are placed in dedicated panels.
- Color is semantic rather than decorative: blue for water/navigation, orange for hazards/range beams, green for debris/detections and navy for reference geometry.
- Artifacts are tested for both existence and resolution.
