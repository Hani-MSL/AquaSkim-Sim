# قرارداد بازتولید یک‌فرمانی

## ورود عمومی

```bat
scripts\bootstrap_and_build.bat
```

## جریان اجرا
1. تشخیص Conda؛
2. ساخت محیط از `environment.yml` در صورت نبودن محیط؛
3. نصب پروژه به‌شکل editable؛
4. Syntax Gate؛
5. پرسش تعاملی دربارهٔ متادیتا، محیط، هندسه، انرژی، مأموریت و اعتبارسنجی؛
6. ساخت `config/user_profile.yaml` محلی؛
7. اجرای تحلیل‌های Phase 02 تا Phase 10.2؛
8. تولید خروجی‌های عددی، تصویری، انیمیشنی، ویدئویی و Evidence؛
9. ثبت SHA-256 و Handoff.

## قرارداد GitHub
کد، تنظیمات پایه، مستندات، تست‌ها و اسکریپت‌ها commit می‌شوند. پروفایل شخصی، خروجی‌ها، Evidence و تحویل نهایی Git-ignored هستند تا هر Clone مستقل بتواند خروجی مخصوص خودش را بسازد.
