# Phase 05 visual-quality standard

Every report figure is exported both as a high-resolution PNG and an SVG.

- Minimum raster resolution: 4500 × 2400 px.
- Equations and long notes are placed in dedicated information panels.
- Plot areas contain only compact labels and legends.
- Tables and raw values are separately exported as UTF-8 CSV.
- The visual-quality manifest records dimensions and paths for every export.
