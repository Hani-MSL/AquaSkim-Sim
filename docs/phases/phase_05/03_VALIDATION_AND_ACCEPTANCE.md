# Phase 05 validation and acceptance

| Check | Criterion |
|---|---|
| Battery usable energy | Positive and lower than nominal energy |
| Pack current | Below configured continuous-current limit |
| Calm cruise endurance | At least configured minimum duration to RTH boundary |
| Calm 60-minute mission SOC | At least configured minimum SOC |
| Return envelope | All analyzed return cases propulsion-feasible |
| Policy consistency | Configured RTH floor is no lower than worst energy-only requirement |
| Visual exports | PNG at least 4500 × 2400 px plus SVG |

All checks are stored as CSV and JSON and recorded in the official phase evidence package.
