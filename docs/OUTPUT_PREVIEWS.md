# Output previews

The repository keeps only lightweight documentation assets under `assets/previews/`. Full-resolution figures, GIFs and MP4 files are generated locally under `outputs/` by the one-command rebuild.

Expected generated output after running the rebuild:

```text
outputs/reports/AquaSkim-Sim_Final_Report.docx
outputs/deliverables/AquaSkim-Sim_Final_Delivery_v1.6.21.zip
outputs/figures/
outputs/animations/
outputs/presentation_evidence/
```

The final audit status should report:

```text
DELIVERY_PACKAGE_READY
```
