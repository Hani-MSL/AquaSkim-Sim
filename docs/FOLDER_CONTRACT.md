# قرارداد ساختار پوشه‌ها

| پوشه | مسئولیت |
|---|---|
| `config/` | پارامترهای مرکزی، سناریوهای شبیه‌سازی |
| `src/aquaskim/` | کد هستهٔ پروژه |
| `cad/` | مدل‌های هندسی، FreeCAD و فایل‌های CAD |
| `visuals/` | تولید شکل، انیمیشن و ویدئو |
| `data/` | دادهٔ خام، پردازش‌شده و ساخته‌شده |
| `outputs/` | تصاویر، جدول‌ها، ویدئوها و لاگ‌های خروجی |
| `tests/` | آزمون‌های خودکار |
| `report/` | تولید گزارش Word نهایی |
| `docs/` | فرض‌ها، معادلات، سناریوها و راهنماها |
| `patch_notes/` | خلاصهٔ تغییرات هر Patch |

تمام ماژول‌ها باید مسیرها را از `aquaskim.paths` دریافت کنند؛ استفاده از مسیرهای مطلق داخل کد ممنوع است.
