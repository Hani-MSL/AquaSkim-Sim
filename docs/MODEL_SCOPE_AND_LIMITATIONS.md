# Model scope and limitations

The model is intended for educational engineering analysis of a sheltered-basin, low-speed autonomous catamaran concept.

## In scope

- deterministic reference missions,
- low-current operating-envelope checks,
- payload and maneuver sensitivity studies,
- reproducible report and evidence generation.

## Out of scope

- sea-trial certification,
- high-sea-state wave-response validation,
- onboard current-estimator validation,
- hardware commissioning or safety approval.

The final delivery package records these non-claims explicitly.
