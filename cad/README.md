# CAD

در Patchهای بعدی، کدهای هندسهٔ پارامتریک، مدل مفهومی FreeCAD و خروجی‌های STEP/STL در این پوشه قرار می‌گیرند.

اصل: مدل CAD باید از `config/base_parameters.yaml` تغذیه شود تا تغییر پارامترها، مدل و محاسبات را هم‌راستا نگه دارد.


## Phase 10.2
`cad/generated/` is created from code and contains conceptual OBJ/STL exports. These are Git-ignored and regenerated by the build scripts.
