# Patch 02.1 — اصلاح کیفیت شکل‌های مکانیکی

## علت Patch
بازبینی واقعی خروجی‌های تولیدشده نشان داد که:
- نام‌گذاری اجزا در شکل‌ها روی هم افتاده بود.
- شکل Funnel در نمای بالا از نظر جهت هندسی مناسب نبود.
- خروجی‌های قبلی وضوح کافی برای گزارش Word نداشتند.
- تست‌ها کیفیت ظاهری و رزولوشن شکل‌ها را کنترل نمی‌کردند.

## اصلاح‌ها
- بازطراحی کامل `phase02.py` با پنل‌های اطلاعات جداگانه و شماره‌گذاری Callout.
- افزودن `visual_quality.py` برای سبک یکنواخت، PNG با وضوح بالا و SVG برداری.
- اصلاح جهت هندسی دهانهٔ جمع‌آوری.
- افزودن جدول کلید اجزا.
- افزودن Manifest کیفیت تصویری.
- افزایش تست‌ها برای رزولوشن و SVG.
- خلاصهٔ فنی فاز 02 به فارسی بازنویسی شد.

## فرمان اجرا
```bat
python -m aquaskim.cli phase02
python -m pytest -q
```

## انتظار
پس از اجرا، سه PNG با وضوح بالا و سه SVG در `outputs/figures/` تولید می‌شوند.
